SPFile
service in module ngSharePoint

Description

Provides functionality to manage SharePoint files.

At the moment, not all methods for manage file objects are implemented in ngSharePoint

Documentation are pending

Methods

  • constructor(web, path, fileProperties)

    Instantiates a new SPFile object for a specific SharePoint file in the server. It's possible to specify their properties.

    By default, in document and picture libraries, when you call getListItems or getItemById, by default a ´item.File´ property are created and contains file information.

    Parameters
    ParamTypeDetails
    webSPWeb

    A valid SPWeb object where the file is stored.

    pathstring

    The server relative path of the file.

    filePropertiesobject

    Properties to initialize the object
