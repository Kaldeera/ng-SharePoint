SPFolder
service in module ngSharePoint

Description

Provides functionality to manage SharePoint folders.

At the moment, not all methods for managing folder objects are implemented in ngSharePoint

Methods

  • getFiles(query)

    Gets the collection of all files contained in the folder.

    Parameters
    ParamTypeDetails
    query
    (optional)
    		object

    An object with all query options used to retrieve files.

    Returns
    promise

    promise with an array of SPFile objects.

    Example

    
  SharePoint.getCurrentWeb(function(web) {
	var folder = new SPFolder(web, '/images');
	folder.getFiles().then(function(files) {
      
          angular.forEach(files, function(file) {
          
              console.log(file.Name + ' ' + file.Length);
          });
     });

  });

  • getProperties(query)

    Makes a call to the SharePoint server and collects all folder properties. The current object is extended with the recovered properties.

    For a complete list of folder properties go to Microsoft SharePoint folder api reference

    SharePoint REST api only returns certain folder properties that have primary values. Properties with complex structures like ParentFolder or Files are not returned directly by the api and it is necessary to extend the query to retrieve their values. It is possible to accomplish this with the query param.

    Parameters
    ParamTypeDetails
    query
    (optional)
    		object

    This parameter specifies which folder properties will be extended and retrieved from the server.

    Returns
    promise

    promise with an object with the folder object

    Example

    This example shows how to retrieve folder properties: 

    
  SharePoint.getCurrentWeb(function(web) {

    web.getFolder("/Images").then(function(folder) {

       folder.getProperties().then(function() {

           // at this point we have all folder properties
           window.location = folder.WelcomePage;
       });
    });

  });
